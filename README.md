# پروژه شبکه عصبی MNIST با NumPy

## ساختار پروژه
- `model.py` : تعریف شبکه عصبی و توابع بارگذاری داده
- `train.py` : آموزش شبکه روی MNIST
- `test.py`  : ارزیابی شبکه
- `mnist_train.csv` : دیتاست آموزش
- `mnist_test.csv`  : دیتاست تست

## آموزش و اجرا

1. آموزش:
    ```bash
    python train.py
    ```
2. تست:
    ```bash
    python test.py
    ```
